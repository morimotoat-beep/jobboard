# 研究者求人サイト 方針更新 v1.1（v1指示書への追記・変更）

この文書は「研究者求人サイト_構築指示書_v1.md」の更新差分。v1の内容は基本そのまま有効で、以下の点を変更・追加する。

---

## 1. サイトの方向性（新規追記・最重要）

**コンセプト：「日本と世界の研究者をつなぐ橋」**

全方位のグローバル媒体ではなく、日本語と英語の境界に特化する。

- **外向き**：日本人研究者が海外のポジション（海外ポスドク・海外大学の教員公募等）を探せる
- **内向き**:海外の研究機関や日本国内の国際公募が、日本語圏の人材に届く

ポジショニング：
- JREC-IN（国内×日本語・年間約2万件）とは正面から競合しない
- Nature Careers（英語圏×英語）とも競合しない
- 「日本語⇔英語の架け橋」というニッチを取る

立ち上げ戦略：アカデミアノート（月1.8万PV）の読者を最初の流入源とし、ブログ記事との相互リンクで育てる。求人が少ない初期は、リンク集＋解説コンテンツを価値の中心に置く。

## 2. 対象言語の変更（v1 §8の修正）

- **v1.1で注力**：日本語・英語の2言語。UIも求人の中身もこの2言語で完全に成立させる
- **中国語・韓国語**:UIの枠組み（既に実装済み）は残すが、翻訳の充実・QAは後回しでよい。新規工数をかけない
- スペイン語:引き続き将来枠

## 3. 自動翻訳の追加（v1 §14の変更:Phase2→v1.1に格上げ）

「英語の求人が日本語で読める／日本語の求人が英語で読める」はこのサイトの核体験のため、早期に実装する。

- **方式**：投稿確定時に1回だけ翻訳してDBに保存（表示のたびに翻訳しない）
- **翻訳API**：DeepL API Free（月50万文字無料。研究用語の精度も良い）
- **対象**：タイトルと要約。日⇔英の2方向を優先（中韓はDBカラムだけ用意し、翻訳実行は後日でよい）
- **DBカラム追加**：
  - `title_ja` / `title_en` / `title_zh` / `title_ko`
  - `summary_ja` / `summary_en` / `summary_zh` / `summary_ko`
  - オリジナルは従来どおり `title` / `summary` に保持
- **表示**：UIの言語に対応するカラムを表示。翻訳がまだ無い言語はオリジナル＋言語バッジで表示（v1の挙動のまま）
- **環境変数**：`.env.local` に `DEEPL_API_KEY=` を追加
- **注記表示**：機械翻訳であることを小さく明示（例「この求人は自動翻訳されています。原文：日本語」）

## 4. 外部求人サイトのリンク集ページ（新規追加）

「世界中の求人への入口（ハブ）」としての価値を持たせるページを追加する。

- ページ：`/links`（各言語版）
- 内容:主要な研究者向け求人サイトへの**リンクのみ**を整理して掲載
  - Nature Careers / Science Careers / JREC-IN / EURAXESS / HigherEdJobs / Times Higher Education Jobs など
  - カテゴリ分け（国内・海外・分野別等）と一言説明を付ける
- **重要（法令・規約遵守）**：
  - 他サイトの求人情報（タイトル・要約・締切等）の取り込み・転載・複製は一切行わない
  - JREC-INは利用規約第12条で転載・複製・再配布を明示的に禁止している
  - リンク（トップページ・検索結果ページ・個別求人ページへのリンク自体）は問題ない
  - RSS取り込み・スクレイピングは実装しない

## 5. 立ち上げ期のコンテンツ方針（参考・実装対象外）

サイト本体の機能ではないが、運営方針として記録:

- アカデミアノートに「海外ポスドクの探し方」「海外学振」等の記事を作り、ジョブボードと相互リンクする
- 求人が集まるまではリンク集＋解説コンテンツを価値の中心とする
- アクセスが安定したら研究室・大学に無料掲載を案内する
- 将来の収益オプション:注目求人の上位表示、ブログ紹介記事とのセット等（実装は当面不要）
